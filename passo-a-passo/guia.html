Como criar um repositório e subir arquivos usando o terminal (Windows)

  1. Acessar o GitHub e criar um repositório novo, sem README.
  2. Abrir o terminal (Git Bash ou cmd).
  3. Clonar o repositório com: 
    git clone https://github.com/SEU_USUARIO/meu-projeto.git
  4. Acessar o diretório: 
    cd meu-projeto
  5. Criar um diretório: 
    mkdir passo-a-passo
  6. Acessar o diretório criado: 
    cd passo-a-passo
  7. Criar um arquivo HTML: 
    echo > guia.html
  8. Escrever este passo a passo dentro do arquivo HTML.
  9. Salvar o arquivo.
  10. Adicionar os arquivos para commit: 
    git add .
  11. Fazer o commit com uma mensagem: 
    git commit -m "Adiciona guia passo a passo"
  12. Enviar as alterações para o GitHub: 
    git push origin main
    ou, dependendo do nome do branch: 
    git push origin master

Pronto! Agora o diretório e o arquivo com o guia estão disponíveis no repositório do GitHub.
